# 终端模拟器协议实现规范

目标：协议支持与 macOS 自带终端（Terminal.app，基于 VT100/xterm-256color）对齐。

本文档每条协议均给出**原始字节示例**（用 `\e` 表示 ESC `0x1B`，`\a` 表示 BEL `0x07`）。

---

## 1. 解析器状态机

终端模拟器**严禁**使用字符串匹配或正则表达式，必须实现基于状态的解析器（参考 Paul Williams 的 VT500 状态机模型）。

### 状态列表

| 状态 | 触发条件 | 说明 |
| :--- | :--- | :--- |
| `Ground` | 初始 / 序列结束后 | 普通可打印字符直接输出，C0 控制字符立即执行 |
| `Escape` | 收到 `ESC (0x1B)` | 等待下一个字节确定序列类型 |
| `EscapeIntermediate` | Escape 下收到中间字符 `( ) %` | 等待最终字节，用于字符集切换等 |
| `Csi` | Escape 下收到 `[` | 积累参数字节 `0x30–0x3F` 和中间字节 |
| `Osc` | Escape 下收到 `]` | 积累 OSC 数据，直到 BEL 或 ST 结束 |
| `OscEscape` | Osc 下收到 `ESC` | 等待 `\` (ST 结束符) |

### 参数格式约定

```
ESC [ <前缀> <p1> ; <p2> ; ... <中间> <动作码>
```

- **前缀**：`?`（DEC 私有模式）、`>`（次级查询）。省略时为标准 ANSI。
- **参数**：十进制整数，以 `;` 分隔。省略的参数等同于 `0`，具体默认值由各指令定义。
- **动作码**：单个 ASCII 字母，决定执行哪条指令。

## 2. C0 控制字符 (0x00–0x1F)

在任意状态下均可立即执行，无需完整序列。

| 符号 | 十六进制 | 示例字节 | 描述 | 行为 |
| :--- | :--- | :--- | :--- | :--- |
| **NUL** | `0x00` | `\0` | 空字符 | 忽略 |
| **BEL** | `0x07` | `\a` | Bell | 触发提示音 / 窗口闪烁 |
| **BS** | `0x08` | `\b` | Backspace | 光标左移一格，不超过行首 |
| **HT** | `0x09` | `\t` | Tab | 移至下一制表位（默认每 8 列） |
| **LF** | `0x0A` | `\n` | Line Feed | 换行；到达滚动区底部时上滚一行 |
| **VT** | `0x0B` | — | Vertical Tab | 同 LF |
| **FF** | `0x0C` | — | Form Feed | 同 LF |
| **CR** | `0x0D` | `\r` | Carriage Return | 光标移到当前行行首（列 0） |
| **ESC** | `0x1B` | `\e` | Escape | 切换至 `Escape` 状态，开始转义序列 |

**示例：** shell 输出一行并换行

```
Hello, World!\r\n
```

## 3. ESC 两字节序列

格式：`ESC <字符>`

| 序列 | 示例 | 名称 | 行为 |
| :--- | :--- | :--- | :--- |
| `ESC 7` | `\e7` | DECSC — 保存光标 | 保存光标位置和 SGR 属性 |
| `ESC 8` | `\e8` | DECRC — 恢复光标 | 恢复 ESC 7 保存的状态 |
| `ESC M` | `\eM` | RI — Reverse Index | 光标上移一行；若在滚动区顶端则插入空行（tmux 上翻依赖此） |
| `ESC =` | `\e=` | DECKPAM — 应用小键盘 | 小键盘进入应用模式（发送 ESC O 前缀） |
| `ESC >` | `\e>` | DECKPNM — 普通小键盘 | 小键盘恢复数字模式 |
| `ESC c` | `\ec` | RIS — 全局复位 | 重置终端到初始状态（等同于硬复位） |

### 字符集切换（ESC 中间字节序列）

格式：`ESC <中间> <最终>`，中间字节为 `(` (G0) 或 `)` (G1)。

| 序列 | 示例 | 效果 |
| :--- | :--- | :--- |
| `ESC ( B` | `\e(B` | G0 = ASCII（默认，正常文字） |
| `ESC ( 0` | `\e(0` | G0 = DEC 线图字符集（tmux / vim 框线） |
| `ESC ) B` | `\e)B` | G1 = ASCII |
| `ESC ) 0` | `\e)0` | G1 = DEC 线图字符集 |

**DEC 线图映射（激活 `ESC(0` 后）：**

| 输入 ASCII | Unicode 输出 | 含义 |
| :--- | :--- | :--- |
| `j` (0x6A) | `┘` U+2518 | 右下角 |
| `k` (0x6B) | `┐` U+2510 | 右上角 |
| `l` (0x6C) | `┌` U+250C | 左上角 |
| `m` (0x6D) | `└` U+2514 | 左下角 |
| `n` (0x6E) | `┼` U+253C | 中间交叉 |
| `q` (0x71) | `─` U+2500 | 水平线 |
| `x` (0x78) | `│` U+2502 | 垂直线 |
| `t` (0x74) | `├` U+251C | 左分叉 |
| `u` (0x75) | `┤` U+2524 | 右分叉 |
| `v` (0x76) | `┴` U+2534 | 下分叉 |
| `w` (0x77) | `┬` U+252C | 上分叉 |

**示例：** tmux 绘制一个分割边框中的水平线

```
\e(0qqqqqqqq\e(B
```

## 4. CSI 序列

格式：`ESC [ [前缀] [参数...] [中间] <动作码>`

### 4.1 光标移动

| 序列 | 示例 | 名称 | 行为 |
| :--- | :--- | :--- | :--- |
| `CSI <n> A` | `\e[3A` | CUU — 上移 | 光标上移 n 行（默认 1），不超过滚动区顶端 |
| `CSI <n> B` | `\e[3B` | CUD — 下移 | 光标下移 n 行（默认 1），不超过滚动区底端 |
| `CSI <n> C` | `\e[10C` | CUF — 右移 | 光标右移 n 列（默认 1），不超过行尾 |
| `CSI <n> D` | `\e[5D` | CUB — 左移 | 光标左移 n 列（默认 1），不超过行首 |
| `CSI <n> E` | `\e[2E` | CNL — 下移到行首 | 下移 n 行并移到列 0 |
| `CSI <n> F` | `\e[2F` | CPL — 上移到行首 | 上移 n 行并移到列 0 |
| `CSI <n> G` | `\e[40G` | CHA — 绝对列 | 移动到当前行第 n 列（1-based，默认 1） |
| `CSI <r>;<c> H` | `\e[10;20H` | CUP — 绝对定位 | 移动到第 r 行第 c 列（1-based，默认 1;1） |
| `CSI <r>;<c> f` | `\e[5;10f` | HVP — 水平垂直定位 | 同 CUP |
| `CSI <n> d` | `\e[5d` | VPA — 垂直绝对 | 移到第 n 行，保持列不变（1-based） |
| `CSI s` | `\e[s` | ANSI 保存光标 | 保存当前光标位置（等同 ESC 7） |
| `CSI u` | `\e[u` | ANSI 恢复光标 | 恢复保存的光标位置（等同 ESC 8） |

**示例：** vim 启动时将光标定位到屏幕左上角

```
\e[1;1H
```

### 4.2 擦除

| 序列 | 示例 | 名称 | 行为 |
| :--- | :--- | :--- | :--- |
| `CSI J` / `CSI 0 J` | `\e[J` | ED — 擦除到屏幕末 | 从光标到屏幕右下角 |
| `CSI 1 J` | `\e[1J` | ED — 擦除到屏幕首 | 从屏幕左上角到光标 |
| `CSI 2 J` | `\e[2J` | ED — 清屏 | 擦除全屏（光标不移动） |
| `CSI 3 J` | `\e[3J` | ED — 清滚动缓冲 | 擦除滚动回退区 |
| `CSI K` / `CSI 0 K` | `\e[K` | EL — 擦除到行尾 | 从光标到当前行末尾 |
| `CSI 1 K` | `\e[1K` | EL — 擦除到行首 | 从当前行开头到光标 |
| `CSI 2 K` | `\e[2K` | EL — 擦除整行 | 清除当前整行 |
| `CSI <n> X` | `\e[5X` | ECH — 擦除字符 | 从光标位置原地擦除 n 个字符（不移动光标，不右移内容） |

**示例：** shell 提示符刷新时擦除当前行再重写

```
\r\e[K新提示符内容
```

### 4.3 插入 / 删除

| 序列 | 示例 | 名称 | 行为 |
| :--- | :--- | :--- | :--- |
| `CSI <n> @` | `\e[3@` | ICH — 插入字符 | 在光标处插入 n 个空格，右侧字符右移，超出行尾丢弃 |
| `CSI <n> P` | `\e[3P` | DCH — 删除字符 | 删除光标处 n 个字符，右侧字符左移，末尾补空格 |
| `CSI <n> L` | `\e[2L` | IL — 插入行 | 在光标行插入 n 空行，滚动区内内容下移，底部行丢弃 |
| `CSI <n> M` | `\e[2M` | DL — 删除行 | 删除光标处 n 行，下方行上移，底部补空行 |
| `CSI <n> S` | `\e[3S` | SU — 向上滚动 | 滚动区内容上滚 n 行，底部插入 n 空行（光标不动） |
| `CSI <n> T` | `\e[3T` | SD — 向下滚动 | 滚动区内容下滚 n 行，顶部插入 n 空行（光标不动） |

**示例：** top 命令刷新进程列表：先滚动区上滚，再重写每行

```
\e[2;24r\e[2S
```

### 4.4 SGR — 字符样式与颜色

格式：`CSI <n1>;<n2>;... m`

#### 基础属性

| 参数 | 示例 | 效果 |
| :--- | :--- | :--- |
| `0` | `\e[0m` | 重置所有属性 |
| `1` | `\e[1m` | 粗体 (Bold) |
| `2` | `\e[2m` | 暗淡 (Dim/Faint) |
| `3` | `\e[3m` | 斜体 (Italic) |
| `4` | `\e[4m` | 下划线 (Underline) |
| `5` | `\e[5m` | 闪烁 (Blink) |
| `7` | `\e[7m` | 反色 (Reverse) |
| `8` | `\e[8m` | 隐藏 (Invisible/Conceal) |
| `9` | `\e[9m` | 删除线 (Strikethrough) |
| `22` | `\e[22m` | 关闭粗体/暗淡 |
| `24` | `\e[24m` | 关闭下划线 |
| `27` | `\e[27m` | 关闭反色 |

#### 标准 16 色

| 参数范围 | 说明 |
| :--- | :--- |
| `30–37` | 前景色：黑红绿黄蓝品青白 |
| `40–47` | 背景色：黑红绿黄蓝品青白 |
| `90–97` | 高亮前景色 |
| `100–107` | 高亮背景色 |
| `39` | 恢复默认前景色 |
| `49` | 恢复默认背景色 |

**示例：** 输出红色粗体文字然后重置

```
\e[1;31mError: file not found\e[0m
```

#### 256 色

```
\e[38;5;<n>m    # 前景色，n = 0–255
\e[48;5;<n>m    # 背景色，n = 0–255
```

颜色索引：0–15 标准色，16–231 六面体，232–255 灰阶。

**示例：** 橙色前景（索引 208）

```
\e[38;5;208mwarning\e[0m
```

#### TrueColor（24 位）

```
\e[38;2;<r>;<g>;<b>m    # 前景色
\e[48;2;<r>;<g>;<b>m    # 背景色
```

**示例：** RGB(255,128,0) 橙色前景

```
\e[38;2;255;128;0mtext\e[0m
```

### 4.5 滚动区域

| 序列 | 示例 | 名称 | 行为 |
| :--- | :--- | :--- | :--- |
| `CSI <t>;<b> r` | `\e[5;20r` | DECSTBM | 设置滚动区域为第 t 行到第 b 行（1-based）；光标移到 (1,1) |
| `CSI r`（无参数）| `\e[r` | DECSTBM reset | 恢复全屏滚动区域 |

**示例：** vim 将 2–23 行设为编辑区，保留第 1 行状态栏

```
\e[2;23r
```

### 4.6 设备状态查询与响应

终端必须主动响应这些查询，否则 vim/tmux 等会卡住等待。

| 发送（程序→终端）| 终端应回复 | 说明 |
| :--- | :--- | :--- |
| `\e[6n` | `\e[<row>;<col>R` | DSR/CPR — 报告光标位置（1-based） |
| `\e[c` 或 `\e[0c` | `\e[?1;0c` | DA1 — 主设备属性（VT100） |
| `\e[>c` 或 `\e[>0c` | `\e[>0;0;0c` | DA2 — 次级设备属性 |

**示例：** vim 启动时发送 `\e[6n`，终端必须立即回复，如光标在第 12 行第 1 列：

```
终端回复: \e[12;1R
```

### 4.7 DEC Private Modes (DECSET/DECRST)

格式：`CSI ? <n> h`（开启）/ `CSI ? <n> l`（关闭）

| 模式 | 示例 | 名称 | 行为 |
| :--- | :--- | :--- | :--- |
| `?1` | `\e[?1h` | DECCKM | 应用光标键模式：方向键发 `\eOA/B/C/D` 而非 `\e[A/B/C/D` |
| `?25` | `\e[?25l` | DECTCEM | 隐藏/显示光标 |
| `?47` | `\e[?47h` | — | 切换备用屏幕（不保存/恢复光标） |
| `?1047` | `\e[?1047h` | — | 切换备用屏幕（切回时清除备用屏幕） |
| `?1049` | `\e[?1049h` | — | 切换备用屏幕 + 保存/恢复光标（vim/tmux 标准用法） |
| `?1000` | `\e[?1000h` | X10 鼠标 | 上报鼠标按下/释放事件 |
| `?1002` | `\e[?1002h` | Button 鼠标 | 上报鼠标按下/拖拽 |
| `?1003` | `\e[?1003h` | AnyEvent 鼠标 | 上报所有鼠标移动 |
| `?1006` | `\e[?1006h` | SGR 鼠标编码 | 鼠标事件使用 SGR 格式（支持大坐标） |
| `?2004` | `\e[?2004h` | 括号粘贴 | 粘贴内容包裹在 `\e[200~` … `\e[201~` 中 |
| `?2026` | `\e[?2026h` | 同步输出 | 批量更新期间暂停渲染（忽略即可） |

**示例：** vim 启动/退出序列

```
启动: \e[?1049h\e[?25l\e[2J\e[1;1H
退出: \e[?25h\e[?1049l
```

## 5. OSC 序列

格式：`ESC ] <n> ; <数据> <终结符>`

终结符为 BEL (`\a`) 或 ST (`ESC \`)。

| OSC | 示例 | 功能 |
| :--- | :--- | :--- |
| `0` | `\e]0;My Terminal\a` | 同时设置窗口标题和图标名 |
| `1` | `\e]1;icon\a` | 仅设置图标名 |
| `2` | `\e]2;My Terminal\a` | 仅设置窗口标题 |
| `4` | `\e]4;1;rgb:ff/00/00\a` | 设置调色板颜色（颜色索引 1 = 红色） |
| `7` | `\e]7;file://host/path\a` | 上报当前工作目录（shell 集成用） |
| `8` | `\e]8;;https://example.com\a文字\e]8;;\a` | 超链接 |
| `10` | `\e]10;?\a` | 查询/设置前景色 |
| `11` | `\e]11;?\a` | 查询/设置背景色 |
| `52` | `\e]52;c;<base64>\a` | 读写系统剪贴板 |
| `133` | `\e]133;A\a` | Shell 集成标记（提示符开始/结束） |

**示例：** zsh 在每次显示提示符时设置窗口标题为当前目录

```
\e]0;~/projects/qterm\a
```

## 6. 鼠标事件编码

鼠标事件由终端上报给应用程序。

### 6.1 X10 编码（?1000）

格式：`ESC [ M <button+32> <x+32> <y+32>`

- button：0=左键，1=中键，2=右键，3=释放，32=拖拽修饰
- x/y：列行坐标，从 1 开始，加 32 后编码为单字节
- 最大坐标 223（255 - 32），不支持大终端窗口

**示例：** 左键点击 (5, 10)

```
\e[M%*    （0x20=32, 0x25=37=5+32, 0x2A=42=10+32）
```

### 6.2 SGR 编码（?1006，推荐）

格式：`ESC [ < <button> ; <x> ; <y> M` (按下) / `m` (释放)

- 支持任意坐标，无大小限制
- button：0=左键，1=中键，2=右键，32=拖拽修饰

**示例：** 左键按下在 (120, 50)

```
\e[<0;120;50M
```

左键释放：

```
\e[<0;120;50m
```

## 8. 括号粘贴模式（Bracketed Paste）

开启 `?2004` 后，用户粘贴内容会被包裹：

```
\e[200~<粘贴的文字内容>\e[201~
```

作用：防止粘贴内容中的 `\n` 被当成"按下 Enter"执行。

## 9. 宽字符与组合字符

- **全角字符**（CJK、Emoji）：占用 2 列宽度，存储在两个相邻 cell 中，右侧 cell 标记为"占位"。
- **组合字符**（Combining Marks，如 `a` + `̈` → `ä`）：零宽，附加到前一个 cell，不移动光标。
- **字符宽度判断**：依照 Unicode East Asian Width 属性，Emoji 按 emoji-data 规则处理为宽字符。
