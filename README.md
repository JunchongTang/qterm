# QTerm

QTerm 是一个面向 Qt 生态的终端模拟器组件库，目标是为 Qt Quick 提供高性能、可扩展、可复用的终端控件能力，同时保留与 QWidget 对接的接口空间。

当前设计重点放在三个方向：

1. 终端协议解析与状态建模
2. 面向 QML 的增量渲染能力
3. 面向本地终端与远程终端的统一会话抽象

## 开发目标

* **Qt 6.8** 是当前开发版本基线。
* **QML / Qt Quick** 是主实现路径。
* **QWidget** 不是主要前端，但会在顶层接口设计上保留接入能力。
* **最小化依赖**，聚焦终端模拟控件本身，而不是引入额外的通用运行时框架。

## 核心能力规划

* **终端内核**: 基于 libvterm 维护终端状态、滚动历史和增量更新。
* **会话后端**: 抽象本地 PTY / ConPTY 与远程终端字节流接入。
* **前端适配**: 以 QTermQuickItem 为主，QWidget 通过共享模型适配。
* **渲染优化**: 基于 QSGNode、字形图集和脏区更新实现高频输出下的平滑渲染。
